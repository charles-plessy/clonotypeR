clonotypes {clonotypeR}R Documentation

Sample clonotype data.

Description

Sample TCR clonotypes extracted from 454 reads and stored as a TAB-separated file, to be loaded with the read.clonotypes function.

Usage

data(clonotypes)

Value

lib

Library name (factor).

V

V segment name (factor).

J

J segment name (factor).

read

Sequence read identifier (character).

dna

DNA sequence of the CDR3 region (character).

qual

Quality values for the DNA sequence (character).

pep

Translation of the DNA sequence (character).
